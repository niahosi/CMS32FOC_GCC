# 逆变器非理想与电流采样误差

本文从外部理论笔记迁入，并已按当前 GCC 工程路径和参数做同步；若后续代码与本文冲突，以当前源码为准。

本文讲为什么 FOC 理论公式在真板子上会变形。

理想模型里，SVPWM 给出什么电压，电机相上就得到什么电压；ADC 读到什么，
就是真实相电流。但实际功率级和采样链路都有误差：

```text
MOSFET 导通压降
死区时间
驱动延迟
母线电压波动
低边采样窗口不足
PGA/ADC 恢复时间
采样电阻噪声和地弹
ADC 顺序扫描相间时差
```

这些误差会直接进入：

```text
iu/iv/iw -> id/iq -> PI -> vd/vq -> duty
```

## 1. 死区造成的电压误差

死区用于避免上下桥臂直通。当前工程：

```text
PWM_DEADTIME_TICKS = 32
1 tick = 15.625 ns
deadtime ~= 0.5 us
```

死区的副作用是：实际输出电压不再等于理想 duty 对应的电压。

它的影响和电流方向有关：

```text
电流从相线流出时，死区期间可能走体二极管/续流路径。
电流流入相线时，误差方向可能相反。
```

所以低速、小电压、小电流时，死区误差占比最大。

典型现象：

```text
很小 iq_ref 时电流不线性
低速有抖动或嗡声
id/iq 有周期性纹波
duty 看起来变化了，但电流响应不成比例
```

当前主固件没有死区补偿，所以低速调试时要保守判断。

## 2. MOSFET 和驱动压降

理想逆变器假设上管导通就是接到 VBUS，下管导通就是接到 GND。

实际还有：

```text
MOSFET Rds_on 压降
体二极管压降
驱动上升/下降时间
供电和地回路压降
```

这些误差会表现成相电压不准，尤其在大电流时更明显。

看数据时：

```text
iq_ref 增大后，vq 增大但 iq 增长不成比例
高电流比低电流更容易 v_limited
电机发热、效率低
```

不能只看 `CTRL_CUR_V_LIMIT`，还要知道真实相上可用电压会比理论值低。

## 3. 母线电压波动

当前固件没有实时 `vbus` watch 字段，文档里的：

```text
V_COUNT_PER_V ~= 128 count/V
```

是按 12 V 母线粗估。

如果母线掉到 10 V，同一个电压 count 对应的实际伏特会变化；电流环以为自己输出了
足够电压，实际电机端没有。

典型现象：

```text
带载时更容易 v_limited
速度越高越上不去
同一组 PI 在不同电源下表现不同
duty 贴边但电流不够
```

建议实验记录里始终记录：

```text
VBUS 空载
VBUS 带载
电流目标
速度目标
v_limited
duty 范围
```

## 4. 采样窗口不足

当前低边三电阻采样要在低边导通窗口里采。

当前窗口相关配置：

```text
CS_OPEN_SETTLE_TICK = PWM_DEADTIME_TICKS + 4
CS_TAIL_MARGIN_TICK = 60
```

如果 duty 太贴边，可用窗口会变窄：

```text
采样点靠近开关边沿
PGA 没恢复
ADC 读到尖峰
某些相无法直接采，只能重构或滤波保持
```

典型现象：

```text
duty 接近 32/1568 时电流突然变差
i_sum 变大
id/iq 出现尖峰
速度环突然抖
```

处理顺序：

```text
先降低 vf_voltage / iq_ref / speed_ref。
观察 duty 远离边界后电流是否恢复。
如果恢复，说明高调制区采样窗口或电压余量是关键。
```

## 5. 当前单点采样的取舍

当前 `CS_MULTI_EN = 0`，每个 PWM 周期只取一个动态窗口中心样本。这样不再依赖
A/B 两点平均或差值判定，控制层可以针对每次有效 PWM 采样推进一次快环。

代价是单点对触发位置更敏感，因此 `CS_OPEN_SETTLE_TICK` 和 `CS_TAIL_MARGIN_TICK`
是当前主要的抗开关噪声手段。若电流尖峰增多，应先观察 duty 和采样窗口，再考虑
恢复双点采样；不要先把电流 PI 参数当成采样问题来调。

## 6. ADC/PGA 零漂和温漂

当前零漂通过 PWM 关闭时采样平均得到：

```text
BOARD_CURRENT_OFFSET_SAMPLES = 1024
MOT_ALIGN_CAL_SAMPLES = 128
```

零漂问题会表现成：

```text
PWM 关闭时 iu/iv/iw 不接近 0
i_sum 有固定偏置
id/iq 在零命令时有偏置
速度环低速抖动
```

温度上升后，PGA 和采样电阻都会漂。冷态正常不代表热态仍正常。

## 7. ADC 顺序扫描的相间时差

当前单 ADC 扫描多个通道时，不同相不是严格同一瞬间采到。

这会带来：

```text
高 di/dt 时，相间采样不是同一时刻的电流
重构第三相时误差变大
Clarke/Park 后出现额外噪声
```

这也是为什么当前配置在三相都可采窗口仍默认只采两相：

```text
CS_USE_2PHASE_IN_ALL_WINDOW = 1
```

目的是减少顺序扫描引入的相间时差。

## 8. 从数据判断非理想问题

更像逆变器/采样非理想：

```text
低速小电流不线性
duty 贴边时突然变差
i_sum 随 duty 区域变化
id/iq 高频噪声随 PWM 输出增强
同一 PI 在低电压/高负载下表现明显变差
```

优先看：

```text
iu_cnt / iv_cnt / iw_cnt / i_sum
id / iq
vd / vq
v_limited
duty_u / duty_v / duty_w
pwm_running
```

不要在这些基础误差明显时盲目调大 PI。
