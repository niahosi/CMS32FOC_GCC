# 06 电机参数填写表

> 2026-06-11 提醒：本表中部分早期规划值需要和当前代码同步复核。当前代码已按 `20 kHz PWM`、`64 MHz SystemCoreClock`、`PGA_GAIN = 2`、`ADC_VREF = 3.6 V` 进行基础验证；`magnet_pole_pairs` 已在调试中按 4 对极处理。此前记录的“电机 1 圈对应丝杠 0.02 mm”已由用户撤销，后续不要用它做位置换算。用户已补充电机参数：电阻 `4 ohm`、电感 `19 uH`、反电势常数 `2.16 V/krpm`。但三线星型电机如果是量两根相线，测到的是线间值，实际单相值通常约为一半，后续必须记录测量方式。

本文用于记录 `CMS32FOCAC6` 项目的电机、编码器、电流采样和 FOC 调试参数。拿到电机资料或实测数据后，优先填写“必须参数”。
