## 当前定位

当前工程已经清理为两个主调试入口：

```text
g_cmd         用户可写命令，只放能改的量
g_elementary 运行观察快照，只放看结果的量
```

旧的 `debug_*`、`g_run`、`g_debug`、`g_test`、`App_Test`、`Motor_Diag` 和 MA600 4 字节诊断入口已从正式源码移除。

## 下一步

先继续整定电流环，不进入位置环。建议测试顺序：

1. `g_cmd.run.motor_enable = 1`
2. 等 `g_elementary.motor.state` 进入闭环状态
3. `g_cmd.run.ctrl_mode = MOTOR_CTRL_CURRENT`
4. `g_cmd.current.id_ref = 0`
5. 从 `g_cmd.current.iq_ref = 0, +/-1, +/-2, +/-3` 小步测试
6. 观察 `g_elementary.foc.id/iq/vd/vq/v_limited`、`g_elementary.pwm.duty_u/v/w`、`g_elementary.motor.fault_reason`
7. 如果 `iq` 抖动或 `v_limited` 频繁为 1，先回查电流采样、相序、电角度零点和 PI 参数。

如果 `id_ref = 0` 但 `id` 很大、`iq` 跟不上正负给定，优先启用 `g_cmd.zero_scan_test` 扫描零点，不要先改 PI。
