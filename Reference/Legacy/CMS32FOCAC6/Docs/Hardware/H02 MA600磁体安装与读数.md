# H02 MA600磁体安装与读数

本文根据 `C18221548_磁性角度传感器_MA600GQE-0000-Z_规格书_WJ1591645.PDF` 整理，重点说明 MA600 与磁铁不同安装方式下，读取到的角度数据代表什么。你的当前结构是“MA600 放在空心磁环侧边”，应按侧轴/偏轴安装处理。

## MA600 实际检测的物理量

MA600 的敏感区域位于封装中心附近，横向小于 400 um，垂直方向大约在封装表面下 300 um。它检测的是平行于封装上表面的 XY 平面内磁场方向，也就是 `Bx/By` 合成向量的角度。

因此，MA600 的 SPI 角度输出本质上是“磁场角”。只有在磁体结构和安装位置让磁场角与机械角一一线性对应时，才能把它直接当机械角使用。

默认情况下，从封装顶部看，磁场顺时针旋转时角度增加。方向可用 `RD` 配置，零位可用 `Z[15:0]` 配置。

## 安装方式 1：端轴同轴安装

端轴同轴安装是规格书推荐的最直接方式：把 MA600 放在旋转轴线上，面对一个径向充磁的圆柱磁铁，磁铁随轴旋转。

特点：

- 传感器中心尽量对准磁铁旋转轴。
- 规格书示例为 N35 圆柱磁铁，尺寸约 `5 mm x 2.5 mm`，磁铁表面到封装表面气隙约 `2 mm`。
- 为获得最佳线性度，传感器中心对轴误差建议控制在磁铁外径的 5% 以内。

读数含义：

- 16 bit `angle_raw` 基本直接对应机械绝对角。
- 换算公式为 `angle_deg = angle_raw / 65536 * 360`。
- `Z[15:0]` 用来设置机械零位。
- `RD` 用来设置正方向。
- 32 bit 读取中的多圈计数和速度可按机械转动理解。

适用性：

- 适合轴端有空间放磁铁和芯片的结构。
- 标定量最少，线性最好。

## 安装方式 2：侧轴偏轴安装

如果轴端没有空间，规格书允许把 MA600 放在圆柱磁铁或环形磁铁的侧边，也就是 side-shaft/off-axis 安装。你的“空心磁环侧边安装”属于这一类。

特点：

- 传感器不在旋转轴线上，而是在磁铁外侧或侧面附近。
- 对空心磁环而言，MA600 看到的是磁环局部的径向磁场和切向磁场合成方向。
- 径向磁场最大值 `BRAD` 和切向磁场最大值 `BTAN` 通常不相等。

规格书用下面的比值描述侧轴安装的非线性：

```text
k = BRAD / BTAN
```

当 `k != 1` 时，MA600 输出的磁场角和真实机械角不再线性。误差曲线呈双正弦形。侧轴配置中，如果传感器中心位于磁铁外径之外，通常 `k > 1`。

读数含义：

- 未补偿时，16 bit `angle_raw` 是磁场角，不是线性机械角。
- 随机械转动一圈，读数通常仍会转一圈，但角度快慢不均匀：某些机械区间读数变化更快，某些区间更慢。
- 多圈计数是“磁场角跨过零点”的次数。若未线性化，过零位置仍受磁场角定义影响。
- 速度输出是磁场角速度。未补偿时，它会包含安装几何导致的周期性速度波动，即使机械匀速转动也可能读出波动。

补偿方法：

1. 先调整 BCT。
2. 再做用户 32 点校准。
3. 最后设置零点。

规格书推荐的顺序就是 `BCT -> 用户校准 -> 零点`。

### BCT 侧轴线性化

BCT 用来降低 X 或 Y 方向 TMR 桥的偏置电流，从而补偿径向/切向磁场幅值不一致。

当 `k` 已知时，推荐值为：

```text
BCT = 258 * (1 - 1 / k)
```

规格书示例：

| k | 误差幅值 E | BCT |
| --- | --- | --- |
| 1.0 | 0 deg | 0 |
| 1.5 | 11.5 deg | 86 |
| 2.0 | 19.5 deg | 129 |
| 2.5 | 25.4 deg | 155 |
| 3.0 | 30.0 deg | 172 |
| 4.0 | 36.9 deg | 194 |
| 5.0 | 41.8 deg | 207 |

注意：

- `BCT > 200` 时补偿结果可能更受温度影响。
- 应减少磁场幅值较大的轴向电流。具体用 `ETX` 还是 `ETY` 取决于芯片封装方向和磁环相对位置。
- 如果 `ETX=1` 且 `ETY=1`，两个方向同时减少，线性度等效于不做 BCT。

### 用户 32 点校准

MA600 支持把 32 个等间隔误差修正值写入寄存器 32 到 63。每 11.25 度一个校准点，芯片运行时做线性插值。

校准值定义为：

```text
corr_i = ref_i - out_i
```

其中 `ref_i` 是真实机械角，`out_i` 是 MA600 未校正输出角。每个校准值范围为 `-11.25 deg` 到 `+11.25 deg`，8 bit 有符号编码。

这一步可以用高精度参考编码器做，也可以在恒速旋转下拟合误差谐波。若侧边安装导致误差很大，仅靠 32 点校准可能不够，应先做 BCT。

## 安装方式 3：多极磁环

如果空心磁环不是一对极，而是多极对磁环，MA600 会在一个机械转动周期内看到多个磁场周期。

读数含义：

- `angle_raw` 是磁角或电角，不是完整机械角。
- 若磁环有 `p` 个极对，则机械转一圈时，MA600 磁角会转 `p` 圈。
- 直接把 `angle_raw` 当机械角会得到周期重复的角度。
- 多圈计数也会按磁角过零次数增加，所以机械一圈可能增加 `p` 次。
- 机械角需要结合极对数和连续解包算法换算。

简单关系：

```text
mechanical_angle_mod_360 = magnetic_angle / pole_pairs
```

实际固件中还要处理跨零、方向、零点和多圈展开。

## 本项目空心磁环侧边安装的建议

本项目应先按“侧轴偏轴 + 可能为空心磁环”的最保守模型处理：

1. 先确认磁环是一对极还是多极对。
2. 低速匀速旋转一圈，记录 MA600 原始角度、霍尔状态和参考机械位置。
3. 检查原始角度是否单调、是否一机械圈输出一磁圈。
4. 若一机械圈输出多磁圈，先确认极对数，再决定是否适合做绝对机械角。
5. 若一机械圈输出一磁圈但非线性明显，先估算 `k`，配置 BCT 和 `ETX/ETY`。
6. BCT 后再做 32 点用户校准。
7. 最后设置 `RD` 方向和 `Z[15:0]` 零位。

## 对固件读取的影响

SPI 数据格式不会因为安装方式变化而变化：

- 16 bit 读角度：仍然是 `angle_raw`。
- 32 bit 读取：前 16 bit 仍是角度，后 16 bit 是多圈或速度。
- Mode 0/Mode 3 都支持，MA600 自动识别。

变化的是数据的物理意义：

| 安装方式 | `angle_raw` 含义 | 是否可直接当机械角 | 主要处理 |
| --- | --- | --- | --- |
| 端轴同轴，一对极 | 磁场角，近似等于机械角 | 可以，仍需零点/方向 | 设置 `RD` 和 `Z` |
| 侧边空心磁环，一对极 | 非线性磁场角 | 不建议直接使用 | BCT、32 点校准、零点 |
| 多极磁环 | 周期性磁角/电角 | 不能直接作为 0-360 机械绝对角 | 极对数换算、连续解包、校准 |

## 上板测试清单

1. 读取 16 bit 原始角度，手动慢转一圈，确认是否连续单调。
2. 记录机械一圈内 `angle_raw` 经过零点的次数，判断磁环极对数。
3. 同步记录 Hall 状态，确认 MA600 正方向与电机相序关系。
4. 若机械匀速但速度输出周期性波动，优先怀疑侧边安装非线性。
5. 调整 `RD` 前先不要写 NVM，临时寄存器验证正确后再固化。
6. BCT 和用户校准完成后再设置零点并写 NVM。
