## 板级概览

本板主控为 `CMS32M6513AGE40NB`，QFN40 封装。它不是普通 MCU 外接驱动的方案，而是集成了三相 `3P3N` 功率驱动的电机控制 SoC。三相功率输出直接接到电机接口，三相低边各串一个 `80 mOhm` 采样电阻到地。

关键资源：

- 内核：Arm Cortex-M0+，最高 64 MHz。
- 存储：64 KB Flash，1 KB Data Flash，8 KB SRAM。
- 功率级：集成三相 `U/V/W` 输出和 `NU/NV/NW` 低边源极。
- 控制外设：EPWM、ADC、PGA0/1/2、ACMP0/1、CCP、DIVSQRT。
- 通信接口：SWD、UART0、SPI、I2C 复用功能。
- 该型号订购表标注驱动供电范围为 `5-18 V`，内部 LDO 为 `5 V`。

## 外部器件

### 霍尔与 NTC

`sub` 页有三颗 `MT9103DT-1616` 霍尔传感器。每颗由 `VCC` 供电，电源旁有 100 nF 去耦，输出端有 4.7 nF 到地滤波。输出分别为 `LHALL_1`、`LHALL_2`、`LHALL_3`。

NTC 分压为 `VCC -> R1 390 ohm -> NTC -> R2 B57421V2222J062 -> GND`，NTC 节点并 C5 100 nF 到地。温度换算不能只靠原理图，需要 NTC 的 B 值/阻值曲线和 ADC 参考电压。

### MA600 磁编码器

`U9 = MA600GQE-0000-Z` 作为 SPI 外设连接：

- `CNS` -> `CS#`
- `MOSI` -> `COPI`
- `MISO` -> `CIPO`
- `SCK` -> `SCLK`
- `AVDD`、`DVDD` -> `VCC`
- `TEST` 和裸露焊盘 -> GND
- `C16 = 100 nF` 从 VCC 到 GND

MA600 支持 SPI Mode 0 和 Mode 3，自动识别。16 个 SCLK 读角度，MSB first。角度换算：

```text
angle_deg = angle_raw / 65536 * 360
```

32 个 SCLK 读取时，前 16 bit 仍是角度，后 16 bit 由寄存器 `MTSP` 决定：`MTSP=0` 为多圈计数，`MTSP=1` 为速度。速度为有符号 16 bit，比例为约 `5.722 rpm/LSB`。

MA600 检测的是封装上表面平行的 XY 平面内磁场方向，不是直接检测机械轴。安装方式会影响“读到的角度”代表什么，详见 [[H02 MA600磁体安装与读数]]。

### 电源

板上 `U8 = TPMP2359DJ` 从 `VM` 降压生成 `VCC`，主要器件为 `L1 = 3.3 uH`、`C13/C14/C15 = 22 uF`，反馈分压 `R9 = 33 kohm`、`R10 = 10.5 kohm`。实际 `VCC` 必须上板测量后确认。

## 推荐初始化序列

这个序列按首次上电调试的保守思路设计：先让功率级保持关闭，再初始化时钟、GPIO、模拟链路、保护链路，最后才允许 PWM 输出。

### 1. 复位入口与时钟

1. 进入启动代码后保持电机输出关闭。
2. 按厂商启动代码要求配置高速内部时钟和 Flash 等待状态。
3. 逐项打开 GPIO、EPWM、ADC、PGA、ACMP、SPI、UART、Timer、WDT 等外设时钟。
4. 开发阶段保留 SWD，不要过早禁用 `P06/P07` 的调试功能。

### 2. GPIO 安全态

1. 先把 `CNS/P02` 配成 GPIO 输出并拉高，再启用 SPI。
2. `P21/P22/P23` 配为霍尔数字输入。
3. `P20` 配为 NTC 模拟输入。
4. `P00/P01/P24/P25/P26/P27` 配为电流采样模拟/PGA 输入。
5. `P06/P07` 开发阶段优先保留 SWD；如果要用作生产 UART，必须先确认烧录和救砖方案。

### 3. 功率级保持关闭

1. 打开 EPWM 外设时钟。
2. 所有 EPWM 输出保持 mask 或 inactive 电平。
3. 在允许输出前先配置刹车保护。
4. 慎用硬件 `BKIN`：本板 `P00/P24/P26` 同时是电流采样脚，不应随意改成数字刹车输入。
5. 在输出仍关闭时配置 PWM 周期、中心对齐、互补模式、极性、死区、加载时刻。

### 4. 模拟参考、PGA、ADC

1. 启用 ADCLDO/参考源并等待稳定。
2. 配置 PGA 伪差分采样：
   - PGA0：`A0P=P00`，`A0GND=P01`
   - PGA1：`A1P=P24`，`A1GND=P25`
   - PGA2：`A2P=P26`，`A2GND=P27`
3. 先用低 PGA 增益，确认采样范围后再提高。PGA 支持 1x、2x、2.5x、5x、7.5x、10x、15x。
4. ADC 通道建议：
   - 使用 PGA 输出时：PGA0 对应 `AN0/AN1`，PGA1 对应 `AN2`，PGA2 对应 `AN3`。
   - 直接采样电阻时：U/V/W 分别可用 `AN8/AN14/AN16`，但信号很小。
   - NTC 使用 `AN4`。
5. EPWM 触发 ADC 采样前，先固定 PWM 时序，采样点避开开关边沿。
6. 电机输出关闭时采集三相零电流偏置，启动时刷新。

### 5. 比较器与过流保护

1. 在 ADC/PGA 偏置确认后再配置 ACMP 阈值。
2. 确认比较器极性后，再接入 EPWM `BRKCTL`。
3. 先在输出 mask 状态测试 `BRKIF/BRKAF`、清除流程和恢复模式。
4. 确认无误后再让刹车链路控制真实 PWM 输出。

### 6. MA600 SPI 编码器

1. `CNS` 默认保持高电平。
2. 配置 `P03/P04/P05` 为 `SCK/MISO/MOSI`。
3. SPI 可用 Mode 0 或 Mode 3，MA600 自动识别；数据 MSB first。
4. 电机输出关闭时先读 16 bit 角度，检查数据是否随转子转动连续变化。
5. 对侧边空心磁环安装，读到的原始角度是磁场角，不能直接假定为机械角。需要做方向、零点、BCT 和用户校准。

### 7. 霍尔接口

1. 先用 GPIO 读取 `LHALL_1/2/3`。
2. 断电手动转动电机，确认霍尔状态稳定且只有 6 个有效状态。
3. 若使用 EPWM 霍尔检测，按用户手册接入 CCP 捕获滤波后的状态。EPWM 支持 `6-2-3-1-5-4-6` 和 `6-4-5-1-3-2-6` 两种序列，其他序列会进入 `111` 错误状态并置位 `HALLIF`。

### 8. PWM 分阶段启用

1. 首次测试建议电机断开，或 `VM` 使用限流电源。
2. 输出 mask 状态下启动 EPWM，确认计数器和加载事件。
3. 用示波器确认 U/V/W 输出极性、互补关系、死区和 inactive 电平。
4. 确认 ADC 采样点和电流读数合理。
5. 刹车、过流、零偏、波形都确认后，再做低占空比开环测试。

### 9. 运行时顺序

1. 快环：EPWM 触发 ADC -> 读电流 -> 扣零偏 -> 控制/换相算法 -> 更新 PWM buffer -> 在安全加载点生效。
2. 慢环：读取 NTC、Hall/MA600 状态、通信状态、故障日志。
3. 故障路径：立即 mask 或 brake EPWM，锁存故障原因，停止控制积分，按明确恢复策略复位。
