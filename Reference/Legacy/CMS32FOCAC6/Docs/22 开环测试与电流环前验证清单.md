# 22 开环测试与电流环前验证清单

本文用于把 VF/IF 开环能测试的内容一次性走清楚，再决定是否进入闭环电流环、速度环。核心原则是：

```text
先用开环把功率级、相序、电流采样、采样窗口和保护确认清楚。
再进入闭环电流环。
不要一开始同时拉高 PWM 占空比、电流环频率和 iq。
```

## 1. 当前工程基线

当前测试以这些配置为准：

```c
#define PWM_FREQ_HZ 20000U
#define PWM_PERIOD 1600U
#define PWM_DUTY_MIN 200U
#define PWM_DUTY_MAX 1400U

#define CURRENT_SAMPLE_PAIR CURRENT_SAMPLE_VW
#define CURRENT_SAMPLE_MULTI_ENABLE 1U
#define CURRENT_SAMPLE_SCORE_ENABLE 1U

#define MOTOR_FAST_LOOP_DIV 2u
```

含义：

```text
PWM 频率：20 kHz
PWM 计数：0 -> 1600 -> 0
当前电流环调度：20 kHz / 2 = 约 10 kHz
当前主采样：固定 physical V/W 双点采样
当前采样评分：低频 KCL 窗口评分层已存在
```

当前控制模式编号：

```text
MOTOR_CTRL_OFF     = 0
MOTOR_CTRL_CURRENT = 1
MOTOR_CTRL_SPEED   = 2
MOTOR_CTRL_VF      = 3
MOTOR_CTRL_IF      = 4
```

开环模式和闭环模式共用底层采样链路：

```text
EPWM CMP0/CMP1 -> ADC -> Board_UpdateCurrent()
```

区别是：

```text
VF：主要用电流做保护，不做 id/iq 闭环。
IF：使用开环 theta 做 id/iq PI。
闭环电流/速度：使用 MA600 电角度做 FOC。
```

## 2. 测试前准备

上电前确认：

```text
VM 使用限流电源。
先低压，不要直接上目标母线电压。
保留 SWD：不要复用 P06/P07。
Ozone/J-Link 已连接，能看到 g_cmd 和 g_elementary。
手边有快速断电手段。
```

测试前先写入安全命令：

```text
g_cmd.run.motor_enable = 0
g_cmd.run.ctrl_mode = 0

g_cmd.open_loop.ol_speed_ref = 400
g_cmd.open_loop.vf_voltage = 40
g_cmd.open_loop.if_id_ref = 0
g_cmd.open_loop.if_iq_ref = 0
g_cmd.open_loop.ol_timeout_ms = 30000
```

不建议第一轮做这些动作：

```text
不要直接 PWM_DUTY_MIN/MAX = 0/1600。
不要直接 if_iq_ref = 80 或更高。
不要同时开启 20K 电流环、高占空比、评分层压力测试。
不要在 J-Link 已经掉线时继续加电流。
```

## 4. VF 开环测试

VF 只输出旋转电压矢量，适合先看功率级、相序、方向和采样。

启动前写：

```text
g_cmd.run.motor_enable = 0
g_cmd.run.ctrl_mode = 0

g_cmd.open_loop.ol_speed_ref = 400
g_cmd.open_loop.vf_voltage = 40
g_cmd.open_loop.ol_timeout_ms = 30000
```

启动：

```text
g_cmd.run.ctrl_mode = 3
g_cmd.run.motor_enable = 1
```

逐步增加电压：

```text
vf_voltage = 40
vf_voltage = 60
vf_voltage = 80
vf_voltage = 100
vf_voltage = 120
```

逐步增加开环速度：

```text
ol_speed_ref = 400
ol_speed_ref = 800
ol_speed_ref = 1500
ol_speed_ref = 3000 
不能转了
```

反向测试：

```text
ol_speed_ref = -400
ol_speed_ref = -800
ol_speed_ref = -1500
-2500就不转了
```

VF 合格现象：

```text
电机能顺滑拖动。
正反转方向可重复。
三相 raw 电流有规律变化。
raw_sum 不长期偏大。
sample_diag_spread_v/w 不长期偏大。
current_over_count 不持续增加。
J-Link 不掉线。
```

VF 异常现象和调整：

| 现象 | 优先判断 | 调整 |
| ---- | ---- | ---- |
| J-Link 一开输出就掉 | 功率级干扰、复位、过流或对齐/输出太猛 | 降低 VM，降低 `vf_voltage`，先保持 `CURRENT_SAMPLE_SCORE_ENABLE=0U` 复测 |
| 电机只抖不转 | 电压太低、负载大、相序/方向异常 | 小幅增加 `vf_voltage`，反向测 `ol_speed_ref` |
| 某一相 raw 特别尖 | 采样通道、PGA、采样窗口或续流路径异常 | 看 `raw_sum` 和 `sample_diag_spread_v/w`，必要时扫 `PWM_ADC_TRIGGER_TICK_DEFAULT` |
| `current_over_count` 增加 | 真实过流或采样毛刺 | 降低 `vf_voltage`，收回 PWM 限幅，检查 raw 电流 |
| `fallback_used` 频繁为 1 | 主 VW 采样窗口不干净 | 暂停加电流，先扫采样点或关闭评分层对比 |

## 5. VF 下看三相 raw 电流

开环慢速转动时重点看：

```text
iu_raw_cnt
iv_raw_cnt
iw_raw_cnt
raw_sum
```

理想趋势：

```text
三相 raw 电流应随开环 theta 有规律变化。
三相之间应有相位关系，不应某一相长期不动或特别尖。
raw_sum 应围绕 0 附近变化，不应长期偏大。
正反转时电流幅值和噪声大体对称。
```

注意当前控制用 logic 映射仍是：

```text
logic Iu = raw V
logic Iv = raw W
logic Iw = -raw V - raw W
```

所以判断硬件采样时，优先看 physical raw 三相；判断控制输入时，再看 logic `iu/iv/iw`。

## 6. IF 开环测试

IF 使用开环 theta 做 Park/InvPark，中间加入 id/iq PI。它比 VF 更接近电流环验证，但仍不依赖 MA600 电角度闭环。

启动前写：

```text
g_cmd.run.motor_enable = 0
g_cmd.run.ctrl_mode = 0

g_cmd.open_loop.ol_speed_ref = 400
g_cmd.open_loop.if_id_ref = 0
g_cmd.open_loop.if_iq_ref = 10
g_cmd.current.kp = 4
g_cmd.current.ki = 0
g_cmd.current.v_limit = 300
```

启动：

```text
g_cmd.run.ctrl_mode = 4
g_cmd.run.motor_enable = 1
```

逐步增加：

```text
if_iq_ref = 10
if_iq_ref = 20
if_iq_ref = 40
if_iq_ref = 60
```

反向测试：

```text
if_iq_ref = -10
if_iq_ref = -20
if_iq_ref = -40
```

IF 合格现象：

```text
小 if_iq_ref 下电机能稳定输出力矩。
正负 if_iq_ref 现象大体对称。
foc.vq 随 if_iq_ref 增大而增大。
foc.v_limited 不长期为 1。
id 不大幅乱飘。
current_over_count 不持续增加。
```

IF 异常现象和调整：

| 现象 | 优先判断 | 调整 |
| ---- | ---- | ---- |
| `v_limited` 长期为 1 | 电压不够、PI 输出顶限、给定太大 | 降低 `if_iq_ref`，降低速度，确认 VM 电压 |
| `id` 大幅漂 | 开环 theta、相序、采样映射或电角关系不对 | 回到 VF 看 raw 三相和方向 |
| `iq` 正负不对称 | 采样窗口、相序、死区/续流路径不对称 | 对比正反 `raw_sum`、`spread`、`fallback_used` |
| 过流计数增加 | 真实电流大或采样尖峰 | 降低 `if_iq_ref`，收回 PWM 限幅，必要时关闭评分层 |

## 7. PWM 占空比逐步放开

当前基线：

```c
#define PWM_DUTY_MIN 200U
#define PWM_DUTY_MAX 1400U
```

推荐压力测试顺序：

```c
// 第一档
#define PWM_DUTY_MIN 150U
#define PWM_DUTY_MAX 1450U

// 第二档
#define PWM_DUTY_MIN 100U
#define PWM_DUTY_MAX 1500U

// 第三档，短时间压力测试
#define PWM_DUTY_MIN 50U
#define PWM_DUTY_MAX 1550U
```

不建议第一轮直接：

```c
#define PWM_DUTY_MIN 0U
#define PWM_DUTY_MAX 1600U
```

原因：

```text
三低边采样需要低边有效窗口。
极高占空比会压缩甚至消灭可用采样窗口。
死区、bootstrap、续流路径和母线尖峰风险都会上升。
完全放开 PWM 只适合低压、空载、短时间、示波器确认后的极限测试。
```

每放开一档都要重新看：

```text
sample_diag_spread_v/w
sample_score_raw_sum_a/b
sample_score_valid_mask
sample_score_fallback_used
current_over_count
J-Link 是否稳定
```

## 8. 10K 与 20K 测试

当前 10K 基线：

```c
#define MOTOR_FAST_LOOP_DIV 2u
```

含义：

```text
20 kHz PWM/ADC 同步节拍 / 2 = 约 10 kHz 控制更新
```

10K 测试建议：

```text
先测 VF。
再测 IF。
确认 J-Link 不掉、过流不涨、采样 spread 不恶化。
```

20K 压力测试：

```c
#define MOTOR_FAST_LOOP_DIV 1u
```

20K 第一轮建议同时关闭评分层：

```c
#define CURRENT_SAMPLE_SCORE_ENABLE 0U
```

原因：

```text
20K 下每个有效采样周期都跑控制。
评分层会插入诊断采样周期，可能让观察变复杂。
先确认主 VW 双点采样和开环输出在 20K 下稳定，再打开评分层复测。
```

20K 起步测试：

```text
VF: vf_voltage = 40 / 60，ol_speed_ref = 400 / 800
IF: if_iq_ref = 10 / 20
```

20K 异常处理：

| 现象 | 调整 |
| ---- | ---- |
| J-Link 掉线 | 回到 10K，降低 VM 和输出，确认是否是功率干扰 |
| `current_over_count` 增加 | 降低 `if_iq_ref`，收回 PWM 限幅 |
| `sample_diag_spread_v/w` 变大 | 回到 10K，扫 `PWM_ADC_TRIGGER_TICK_DEFAULT` |
| `fault_reason` 出现角度故障 | 检查 MA600 读角负载，先不要继续加频率 |

## 9. 评分层复测

主采样稳定后再打开：

```c
#define CURRENT_SAMPLE_SCORE_ENABLE 1U
#define CURRENT_SAMPLE_SCORE_INTERVAL_PWM 10U
```

重点观察：

```text
sample_score_raw_sum_a
sample_score_raw_sum_b
sample_score_valid_mask
sample_score_best_window
sample_score_fallback_used
```

合格现象：

```text
至少部分窗口能得到有效评分。
raw_sum_a/b 不长期偏大。
fallback_used 不频繁为 1。
打开评分层后，VF/IF 现象不明显变差。
```

异常调整：

```text
如果 win0/1/2 长期无效：窗口可能太窄，先收回 PWM 占空比限幅。
如果 fallback_used 频繁为 1：主采样窗口不干净，先扫采样点。
如果打开评分层后 J-Link 或 fault 变差：先关闭评分层，把主路径调稳。
```

## 10. 总体验收标准

可以认为“开环基础通过”的条件：

```text
VF 正反转顺滑。
三相 raw 电流有规律。
raw_sum 不长期偏大。
IF 小电流正负都能运行。
id 不大幅乱飘。
v_limited 不长期为 1。
current_over_count 不持续增加。
J-Link 不掉线。
采样 spread 和评分层没有明显恶化。
```

未通过时不要进入闭环电流环，优先按这个顺序收回：

```text
1. 降低 vf_voltage 或 if_iq_ref。
2. 收回 PWM_DUTY_MIN/MAX 到 200/1400。
3. 关闭 CURRENT_SAMPLE_SCORE_ENABLE。
4. 回到 10K：MOTOR_FAST_LOOP_DIV = 2u。
5. 降低 VM，确认 J-Link 和电源稳定。
6. 回到 VF 慢速重新看 raw 三相。
7. 必要时扫 PWM_ADC_TRIGGER_TICK_DEFAULT。
```
