## 8. 当前代码状态摘要

截至 2026-06-10：

- 自写源码路径：`User/App`、`User/Board`、`User/Motor`；Keil group 名仍保留 `App / Board / Motor`。
- 可调参数入口：`User/Config/Config.h`，集中电机/编码器参数、控制环参数、保护阈值、PWM 频率和开环默认值。
- 编译主线：Keil MDK + ARMCC5。
- 系统时钟：`64 MHz`。
- PWM：`20 kHz`，三相 PWM 已能在示波器上看到。
- ADC：EPWM CMP0 触发两相电流同步采样，第三相由三相电流和重构，ADC 中断进入 `Motor_FastLoop()`。
- MA600：能读取角度；当前按 4 对极磁环处理，FOC 不再额外乘 4。
- Motor：已有 `MOTOR_CTRL_CURRENT`、`MOTOR_CTRL_SPEED`、`MOTOR_CTRL_VF`、`MOTOR_CTRL_IF`，当前测试配置下电流环约 `10 kHz`，速度环分频需随此重新核对。
- 程序已清理为“正式 FOC 主线 + 开环检测辅助”。
- 调试写入口：`g_cmd`；观察入口：`g_elementary`。
- 已从正式工程移除 `App_Test`、`Motor_Diag`、`Board_Test`、AutoZero、旧 `current_loop_req`、旧 `debug_*`、`g_run/g_debug/g_test` 和 MA600 4 字节诊断入口。
- 位置环：只写设计规划，暂不启用。
